## Installation

### Bower

```sh
$ bower install ovh-module-sharepoint --save
```

### NPM

```sh
$ npm install ovh-module-sharepoint --save
```

## Get the sources

```sh
$ git clone https://github.com/ovh-ux/ovh-module-sharepoint.git
$ cd ovh-module-sharepoint
$ npm install
$ bower install
```
